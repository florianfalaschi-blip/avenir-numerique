# @avenir/db

> Types Supabase générés, clients typés, repositories de données.

## Périmètre

- Types TypeScript auto-générés depuis le schéma Supabase
- Client Supabase configuré (côté serveur et navigateur)
- Repositories pour chaque entité (devis, clients, commandes, etc.)

## Génération des types

```bash
pnpm db:generate
```

(à configurer avec `supabase gen types typescript`)
