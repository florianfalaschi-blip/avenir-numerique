# apps/admin — Back-office Avenir Numérique

> Application back-office pour la gestion commerciale, devis, production, clients et facturation.

## Fonctionnalités prévues

- 5 calculateurs métier (Roll-up, Plaques, Flyers, Bobines, Brochures)
- Gestion devis (création, validation, refus, archivage)
- Portefeuille clients (B2B / B2C) avec carnet d'adresses
- Planning production avec Gantt drag & drop
- Réception commandes web (workflow BAT)
- Module facturation interne (PDF, exports comptables, refunds)
- Reporting / statistiques

## Stack

- Next.js 15 App Router
- shadcn/ui + Tailwind CSS
- Supabase (auth, data, realtime)
- React Hook Form + Zod
- TanStack Query

## Initialisation

Ce projet sera initialisé en Phase 2 par le prestataire.

Commande type :
```bash
pnpm create next-app@latest . --typescript --tailwind --app
```
