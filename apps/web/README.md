# apps/web — Site e-commerce Avenir Numérique

> Site e-commerce public B2B + B2C.

## Fonctionnalités prévues

- Catalogue produits + configurateur (grille pré-calculée + sur-mesure)
- Authentification B2B / B2C (validation manuelle B2B)
- Panier + checkout Stripe (CB / PayPal / Virement / 30j fin de mois B2B)
- Upload fichiers d'impression + gabarits téléchargeables
- BAT optionnel payant
- Espace client (commandes, factures, BAT, adresses, ré-achat)
- Suivi commande avec timeline
- Newsletter + module avis + parrainage + codes promo

## Stack

- Next.js 15 App Router (SSR/SSG pour SEO)
- shadcn/ui + Tailwind CSS
- Supabase (auth, data, realtime)
- Stripe Checkout
- React Hook Form + Zod

## Initialisation

Ce projet sera initialisé en Phase 2 par le prestataire.
